2 - Do a FOR from 1 to 100 with index i
3 - IF the i is a multiple of three and five 
        - Concatenate an element(including box both-box classes and text: 'FizzBuzz') to the container
        
    ELSE IF i is a multiple of three
        - Concatenate an element(including box fizz-box classes and text: 'Fizz') to the container
    
    ELSE IF i is a multiple of five
        - Concatenate an element(including box buzz-box classes and text: 'Buzz') to the container
    
    ELSE
